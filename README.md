# RollingGourmeteBack

Repositorio del frontend de Rolling Gourmet

## Tabla de contenido

- Crud usuarios
- Crud reservas
- Contacto
- Galería
- Nosotros
- Reservas
- Registro
- Login

El proyecto utiliza una base de datos remota en MongoDB atlas y un backend en Render
